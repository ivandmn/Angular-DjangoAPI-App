1. Create project -> python -m django startproject "project_name"
2. Create app -> python -m django startapp "app_name"
3. Add models of existing database -> python manage.py inspectdb > api/models.py
4. Make migrations from models.py -> python manage.py makemigrations
5. Migrate -> python manage.py migrate
6. Create superuser -> python manage.py createsuperuser
7. Open Server in localhost -> python manage.py runserver
